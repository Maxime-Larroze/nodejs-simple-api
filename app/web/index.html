Simple API NodeJS

Routes

/ (get)

Cette interface d'informations

/users (get)

Retourne tous les utilisateurs inscrits dans la base de données

/users (post)

Créé un utilisateur dans la base de données et retourne son ID

/users/{id} (get)

Retourne toutes les informations d'un utilisateur (à l'exception du mot de passe) via son ID

/users/{id} (put)

Modifie un utilisateur via son ID

/users/{id} (delete)

supprime un utilisateur via son ID

/documents (get)

Retourne l'URL de tous les documents disponibles sur le serveur

/documents {file, userid} (post)

Upload un document sur le serveur et inscrit l'URL à l'avatar du "userid" passé en paramètre

/documents/{id} (get)

Retourne l'URL du document via son appelation placé en paramètre id

/documents/{id} (delete)

Supprime un document du serveur via son appelation placé en paramètre id
